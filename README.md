# Active Directory User Creation Script

Script em PowerShell para criação de usuários no Active Directory a partir de um arquivo CSV.

O objetivo do script é permitir a importação em massa de usuários de forma simples e adaptável para diferentes estruturas de CSV.

## Requisitos

- Windows Server ou máquina com RSAT instalado
- Módulo ActiveDirectory disponível
- Permissão para criar usuários no AD
- PowerShell 5+
